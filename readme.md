проект API для Avito

Билд и запуск проекта:
- docker-compose up --build

Доступ к api проекта:
- http://127.0.0.1:8080/

Доступ к БД через adminer:
- http://127.0.0.1:8081/
- engine: PostgreSQL
- server: postgres
- user: postgres
- password: password
- database: myapp
